Rotação cíclica

Rotacione um array para a direita, por um número determinado de vezes.

Um array de posição A, consiste de N inteiros acumulados. A rotação do array vai fazer com que cada elemento seja deslocado para a direita de um dos índices, e o último elemento do array, será movido para a primeira posição. Por exemplo, a rotação do array A = [3, 8, 9, 7, 6] é [6, 3, 8, 9, 7] (pois os elementos são deslocados para a direita de cada índice, e 6 é movido para a primeira posição).

O objetivo é rotacionar o array A um número de K vezes, tal que, cada elemento de A seja deslocado para a direita um número de K vezes.

Escreva uma função:

def solution(A, K)

tal que, dada uma matriz A que consiste em N números inteiros e um número inteiro K, retorna a matriz A rotacionada K vezes.

Por exemplo, dado que

A = [3, 8, 9, 7, 6] K = 3

A função deve retornar [9, 7, 6, 3, 8]. Três rotações ficam feitas como:

[3, 8, 9, 7, 6] -> [6, 3, 8, 9, 7] [6, 3, 8, 9, 7] -> [7, 6, 3, 8, 9] [7, 6, 3, 8, 9] -> [9, 7, 6, 3, 8]

Por exemplo, dado que

A = [0, 0, 0] K = 1

A função irá retornarn [0, 0, 0]

Dado que

A = [1, 2, 3, 4] K = 4

A função deve retornar [1, 2, 3, 4]

Assumindo que:

  • N e K são inteiros dentro do intervalo [0..100];
  • cada elemento do array A é um inteiro dentro do intervalo de −1,000..1,000].

Na sua solução, o foco deve ser na exatidão. O desempenho da sua solução não será o foco da avaliação.
