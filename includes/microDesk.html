MicroDesk CRM para suporte em informática

Sobre o projeto:

    A aplicação registra atendimentos e O.S. de empresas de suporte em informática. Criada originalmente para atender clientes de automação comercial. Apesar de ser feito para uma empresa específica, é possível personalizar a logo, e alterar o código fonte do projeto. Compatível com a versão 5.0 do PHP por ser uma aplicação antiga. Não há uma data para atualização da aplicação. Está disponível no github .

Especificações

  • Tipo: Sistema
  • Visibilidade: Pública
  • Linguagens: PHP, MySQL, XML, JavaScript, HTML, CSS
