DuffApp FrameWork FrameWork para criação de aplicativos

Sobre o projeto:

    DuffApp FrameWork é um aplicativo voltado para desenvolvedores, como base para acesso ao hardware do dispositivo, usando HTML5 puro, sem injeção de frameworks visuais. Conta com acesso a câmera, GPS e memória do dispositivo. O projeto está disponível no github, e se você é desenvolvedor, pode testá-lo baixando e gerando o apk usando o cordova.

 

 

Especificações

  • Tipo: Aplicativo
  • Visibilidade: Pública
  • Linguagens: HTML5 (JavaScript/CSS/HTML)
