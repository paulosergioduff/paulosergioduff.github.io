BR Busca JS é um sistema de busca todo em JavaScript, com paginação e exibição de 
resultados. No momento da busca o usuário pode optar por um ou mais banco de dados ou
separar um banco de dados para cada categoria. O sites devem ser cadastrados nos banco de
dados em js, contidos no mesmo diretório, onde são armazenados todos os dados dos sites
cadastrados.
